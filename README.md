# Generate project ()

> This cli-app use to create project base on preprocessor option

## Install

```bash
$ npm i -g @nhn.dev/generate_frontend_project
```

## Usage

```bash
$ genproject init abc
```

This will show some preprocessor that app can create

## Other

**Show help**

```bash
$ genproject --help
```

**Show version**

```bash
$ genproject --version
```

**Show directory tree**

```bash
$ genproject --tree
```

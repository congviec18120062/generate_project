# Generate project ()

> This cli-app use to create project base on preprocessor option

## Install

---

```bash
$ npm install @nhn.dev/generate_frontend_project
```

## Usage

---

```bash
$ gensetup init abc
```

This will show some preprocessor that app can create

## Other

---

**Show help**

```bash
$ gensetup --help
```

**Show version**

```bash
$ gensetup --version
```

**Show directory tree**

```bash
$ gensetup --tree
```
